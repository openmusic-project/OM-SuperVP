OM-SuperVP is a library for the control of SuperVP from OpenMusic.n The library contains functions for sound analysis (FFT, Transients, Fundamental frequency, Formants) and processing (Time/Frequency transformations, filters, cross-synthesis, etc.).nn OM-SuperVP requires SuperVP to be installed, authorized, and declared as an OM external (see Preferences/Extrernals).
